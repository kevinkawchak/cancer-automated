# papers/VVUQ-03 — Visual Representations Catalog

**Source directory:** `papers/VVUQ-03/` (top-level `README.md` plus every subdirectory, recursively).
**Scope:** Every `.md` and `.txt` file under `papers/VVUQ-03/`. Visual types extracted: Markdown pipe tables, ASCII / text diagrams in fenced code blocks, ASCII timelines, and repository / directory structure trees. LaTeX source files (`.tex`, `.sty`, `.bib`) and binaries (`.zip`) are out of scope. NOTE on the exclusion: the `template-paper/chunk_*.md` files reproduce LaTeX source verbatim (they contain `\begin{longtable}`, `\begin{description}`, etc.); those LaTeX tabular/longtable blocks are LaTeX source, not rendered visuals, and are therefore NOT extracted. Plain command-recipe code blocks (e.g. the four-line `pdflatex` "Compile recipe" blocks) are not tables, diagrams, timelines, or directory trees, and are not cataloged. Tab-separated text "tables" in the instruct-paper outputs are not Markdown pipe tables (no `|` delimiters, no `|---|` row) and are not inside fenced code blocks, so they are out of scope.
**Extraction date:** 2026-06-02.
