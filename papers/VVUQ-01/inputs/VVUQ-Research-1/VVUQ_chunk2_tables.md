# Tables — VVUQ in Clinical Trial Data Pipelines and Digital Twins

**Table: Sample VVUQ Methods and Outcomes in Clinical AI Pipelines**

| **Application / Pipeline** | **Method / VVUQ Approach** | **Key Metrics & Outcomes** |
| --- | --- | --- |
| **LLM Data Extraction** UODBLLM (Clinical EHR) | Schema-constrained GPT-4 prompts; JSON schema validation; retry/requeue logic for errors. Modular versioned prompts. Human audit on critical fields. | 98.1% median field accuracy on 424 reports; 95%+ concordance on other test set. 8.9 s per report, 100% completion, $0.009 cost/report. Replaced manual abstraction in production. |
| **LLM Code Generation** Compiled AI (Healthcare Workflows) | Constrain LLM to generate narrow functions; **4-stage validation** of generated code (security, syntax, execution, accuracy); deploy static code with determinism. | 96% function-task completion (384/400) with *no run-time inference*, break-even at ~17 uses. Compiled artifacts executed with 100% reliability once verified. Prompt-injection detection 96.7%, static code safety 87.5%. Significant token/cost savings at scale. |
| **Digital Twin Trials** Sanofi & Phesi (Trial Planning) | Quantitative Systems Pharmacology (QSP) modeling to create virtual patients; statistical and ML methods for synthetic control arm. Early validation via "blind" predictions. | Sanofi's asthma twin model accurately predicted a Phase 1b trial outcome before data release, building sponsor confidence. Phesi's twins incorporate >10× patient data and show consistent outcomes; claim synthetic control arm could entirely replace a physical control. |
| **In Silico PBPK Dosing** Theranostic Digital Twins | Rigorous VVUQ per FDA/ASME: model structure verification; calibration; validation against preclinical/clinical PK data; Monte Carlo and sensitivity analysis for UQ. | Highlighted in review: "reliability must be established through rigorous VVUQ". Formal UQ needed to provide confidence intervals on predicted dosimetry. This foundational work sets stage for regulatory acceptance of PBPK-based treatment planning. |
| **Image-based Models** Cancer Therapy Simulations | Traditional (mesh-based) modeling augmented by ML; validation on surrogate data; Monte Carlo UQ. VVUQ summarized as: verify implementation, validate with internal/external data, quantify prediction variance. | Emphasized that only validated models with UQ should guide interventions. Example metrics include fit-to-imaging data, reproducibility across patients, and uncertainty bounds on dose distribution. Stressed moving beyond qualitative results to clinically credible simulations. |
