# examples-chunking/

Runnable examples for autochunking and reconstruction READMEs.

| Script | Shows |
|--------|-------|
| `01_chunk_document.py` | Chunk a large document and report each chunk size |
| `02_chunk_with_readme.py` | Chunk and generate the reconstruction README |

```bash
python chunking/examples-chunking/01_chunk_document.py
python chunking/examples-chunking/02_chunk_with_readme.py
```
