# examples-simulation/

Runnable examples for triple simulation and consensus.

| Script | Shows |
|--------|-------|
| `01_triple_run.py` | Run a project three times and print the per-run metrics |
| `02_consensus_report.py` | Aggregate the three runs into a consensus report |

```bash
python simulation/examples-simulation/01_triple_run.py
python simulation/examples-simulation/02_consensus_report.py
```
