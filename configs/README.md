## Cancer-Automated configs

#### pipeline_config.yaml
- Daily-deliverable pipeline configuration for cancer-automated.
- Production-ready defaults aligned with the Stage 1 roadmap.

#### vvuq_thresholds.yaml
- VVUQ gate thresholds for cancer-automated.
- VVUQ is held to a higher standard than code generation, so the gate is
- intentionally strict. A deliverable is accepted only when every threshold
- below is satisfied across all three simulation runs.
