# examples-physical-ai/

Runnable examples for the Stage 2 deployment references. These are simulations;
they do not actuate any hardware.

| Script | Shows |
|--------|-------|
| `01_lights_off_cell.py` | Run an autonomous batch and trigger an emergency stop |
| `02_hybrid_pilot.py` | Build the hybrid surgery and medicine pilot timeline |

```bash
python physical-ai/examples-physical-ai/01_lights_off_cell.py
python physical-ai/examples-physical-ai/02_hybrid_pilot.py
```
